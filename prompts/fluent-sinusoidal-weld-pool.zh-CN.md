# 角色
你是会使用 Codex、本地文件系统、Ansys Workbench 2025 R2、DesignModeler/SpaceClaim、Ansys Meshing、Fluent、Fluent UDF、PyFluent、CFD-Post/Fluent 后处理的焊接熔池 CFD 自动化工程师。

# 总目标
在本机使用 Ansys Fluent 建立“正弦轨迹移动热源焊接熔池”分阶段验证模型，模拟 AISI 1045 钢平板焊接过程中的温度场、液相分数、熔池形态、熔池深度随时间变化、正弦焊接轨迹、MP4 动画和熔池参数曲线。

本任务优先目标是完成可执行、可诊断、可逐步加密的工程验证流程，而不是一次性得到报告级最终结果。

# 几何与坐标
名义工件：100 mm x 30 mm x 10 mm。
第一阶段可使用局部短焊道验证模型，例如 30 mm x 20 mm x 6 mm。

坐标定义：
- X：焊接前进方向；
- Y：正弦横向摆动方向；
- Z：厚度方向；
- 热源作用在上表面；
- 重力方向必须根据实际模型坐标确认后再设置。

不要默认使用长度或宽度方向对称面。不要预先把液态金属域和固态金属域切成固定区域，固液相变应由 Fluent Solidification/Melting 的液相分数追踪。

# 分阶段策略
阶段 A：热传导 + Solidification/Melting。
1. 开启 Energy。
2. 开启 Solidification/Melting。
3. 使用焓-孔隙率法。
4. Mushy Zone Parameter 初始取 1e5，并说明敏感性。
5. 暂不启用 VOF，不考虑自由表面变形。
6. 验证移动正弦热源、温度场、液相分数和熔池深度变化。

阶段 B：加入熔池内流动。
1. 开启层流模型。
2. 启用重力和浮力。
3. 根据坐标确认重力方向。
4. 使用 Boussinesq 近似时说明温差风险。
5. Marangoni 效应优先作为高级边界或源项处理。

阶段 C：VOF + 保护气体 + 自由表面。
只有阶段 A/B 成功后才进入。设置气体区域、VOF、表面张力和温度相关表面张力。dγ/dT 初始范围取 -0.0003 到 -0.0005 N/(m K)。必须说明 VOF 自由表面结果可信度。

# 网格
采用分区网格策略。阶段 A 熔池附近局部网格建议 0.2-0.5 mm；短焊道验证成功后才细化到 0.05-0.1 mm。记录网格数量、最小尺寸、最大 skewness、最大 aspect ratio。如果 skewness > 0.8 或网格数量过大，应停止并报告。

# 材料参数
材料：AISI 1045 钢，全部使用 SI 单位。
- 密度：7850 kg/m3
- 固相线温度：1420 °C
- 液相线温度：1460 °C
- 熔化潜热：250000 J/kg
- 比热容：470-480 J/(kg K)
- 导热系数：室温约 48-51 W/(m K)
- 液态金属动力粘度：0.005-0.01 kg/(m s)
- 热膨胀系数：1.15e-5 1/K

未验证材料参数必须标记为工程估计，不要称为文献标定值。

# 正弦移动热源
第一阶段优先使用移动高斯表面热流。
- Q = 10000 W，若发散可降到 1000 W 做验证
- eta = 0.75
- v = 0.003 m/s
- A = 0.003 m
- freq = 2.0 Hz
- r_beam = 0.003 m

轨迹：
x_center(t) = v * t
y_center(t) = A * sin(2 * pi * freq * t)

UDF 要求：
1. 使用 DEFINE_PROFILE 定义壁面热流。
2. 使用高斯热流分布。
3. 编译前检查 Fluent 双精度、编译器环境和 UDF 文件路径。
4. 编译失败必须输出完整错误信息。
5. 热源必须体现沿 X 方向前进，同时沿 Y 方向正弦摆动，不能只在原地摆动。

# 求解与后处理
使用瞬态求解。第一轮时间步可取 0.05 s，输出间隔建议 0.1 s。保存 case/data、Fluent 日志、CSV/JSON、图片、MP4 和最终报告。检查 divergence、floating point error、temperature limited、UDF compile error 和 solution failed。

必须输出：
1. 温度云图；
2. 液相分数云图；
3. 熔池截面图；
4. 热源轨迹图；
5. 熔池参数随时间变化曲线图；
6. 实时温度分布云图 MP4；
7. 熔池深度截面变化 MP4；
8. 熔池参数 CSV/JSON。

熔池截面视频建议同时包含固定中心截面 Y=0 和跟随热源截面 Y=y_source(t)。熔池判据优先使用 liquid_fraction >= 0.5，并说明熔深精度受 Z 向网格尺寸影响。
