# COMSOL EHD Soybean Evidence Chain

Use this checklist before claiming that an EHD ion-wind soybean drying model ran
successfully.

- [ ] `COMSOL_ROOT`, `COMSOL_BATCH`, `COMSOL_COMPILE`, and `COMSOL_JAVA` checked.
- [ ] COMSOL API documentation path checked or local docs unavailable noted.
- [ ] Run folder created with `run.json` and `run.log`.
- [ ] Java model source saved under `scripts/`.
- [ ] `comsolcompile` log captured under `logs/`.
- [ ] `comsolbatch` log captured under `logs/`.
- [ ] `.mph` model artifact exists if model generation succeeded.
- [ ] EHD model type is labeled: equivalent body force, equivalent inlet velocity, or real discharge.
- [ ] Material and drying parameters are labeled as source-backed or engineering assumptions.
- [ ] At least one no-EHD and one EHD case are compared.
- [ ] CSV or JSON numeric outputs exported.
- [ ] Plots or screenshots exported.
- [ ] `evidence.json` generated by the toolbox.
- [ ] `report.md` explains failures, assumptions, and next validation steps.
