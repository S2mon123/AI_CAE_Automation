# Sample Outputs

These files are representative outputs from the local Stage A3c validation run.
They are included so readers can inspect the expected evidence shape without
running Fluent first.

Included:

- temperature-cloud MP4,
- melt-pool depth-section MP4,
- melt-pool parameter curve PNG/SVG,
- melt-depth time-series CSV,
- run and postprocessing JSON summaries,
- sinusoidal path evidence.

Not included:

- the 100 dense transient `.dat.h5` files,
- UDF compiled DLL folders,
- local Fluent transcript logs.
