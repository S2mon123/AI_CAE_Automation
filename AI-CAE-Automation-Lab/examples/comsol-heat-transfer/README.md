# COMSOL Heat Transfer Example Skeleton

This folder is a public project layout for a COMSOL heat-transfer validation case.

## Suggested Layout

```text
scripts/
  HeatSinkModel.java
runs/
  <timestamp>/
    heat_sink.mph
    solver.log
    images/
    tables/
    report.md
```

## First Validation Target

- parameterized geometry,
- steady laminar flow,
- heat source,
- temperature and velocity plots,
- maximum temperature table.
